# Git & GitHub Notes

This project contains a simple HTML notes page that explains Git and GitHub from beginner to advanced topics.

## Files

- `index.html` — the notes page with sections on Git basics, GitHub, branching, pull requests, advanced Git, workflows, and a command cheatsheet.
- `style.css` — styling for the notes page.

## How to use

1. Open `index.html` in your browser.
2. Review the guide from the introduction through advanced Git commands.
3. Use this page as a reference while learning Git and GitHub.

## Contents covered

- Git repository setup
- Adding, committing, and viewing history
- Connecting a local project to GitHub
- Branching, merging, pull requests
- Rebasing, interactive rebase, cherry-pick, stash, reset, tags
- GitHub workflows and best practices
- Command cheatsheet for common Git operations

## Notes

This project is meant as a study guide or quick reference for developers learning or reviewing Git and Github.
